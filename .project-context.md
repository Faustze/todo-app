## Ключевые решения

### Тема
- `createThemePlugin` без кастомного адаптера → `--v0-*` на `<html>`
- `colors.ts` — единый источник (23 ключа)
- ThemeToggle: `useTheme().cycle()` + ViewTransition circle-анимация

### Vuetify0 primitives
- Button (variant × size × color), Single (filter), Dialog (modal), Atom
- `storeToRefs()` для Pinia refs, actions напрямую

### DkButton color
- `color?: ThemeColorKey` — TypeScript подсказывает все 23 ключа из colors.ts
- CSS custom properties `--btn-color` / `--btn-on-color` через inline style
- `on-*` fallback: primary→on-primary, muted→on-surface и т.д.

### Валидация
- Vuelidate + `helpers.withMessage()` на русском
- `reactive` (не computed) — v-model работает
- Input.Root `:error` = `v$.field.$error`, Input.Error рендерит `$errors`
- Title: required + minLength(3) + maxLength(200), Description: maxLength(500)

### Фильтрация
- DkToggle defineModel → computed activeFilter → storeToRefs → setFilter
- TaskList через storeToRefs → filteredTasks (reactive)

### Поиск
- pages/index.vue: инпут поиска под фильтрами (DkToggle), v-model → storeToRefs searchQuery
- searchTasks() в task-filters.ts: case-insensitive title filter, после фильтрации по статусу
- filteredTasks = filterTasks() → searchTasks() (цепочка: статус → поиск)

### Сортировка
- DkSortButtons: 2 кнопки (Дата / Приоритет) + toggle direction (asc/desc)
- sortTasks() в task-filters.ts: PRIORITY_ORDER {low:1, middle:2, high:3}, date по createdAtUtc
- Цепочка: filterTasks → searchTasks → sortTasks

### Модалки
- DkModal: Dialog.Root + Content (без Teleport)
- Responsive: center (desktop) / bottom-sheet (mobile, smAndDown)
- DkToggle: `defineModel({ default: 'all' })`, `@click="model = option.value"`

### Select styling
- `appearance:none`, кастокная SVG-стрелка через background-image
- `padding-right: 2.5rem`, стрелка на `right 0.75rem center`
- Focus: стрелка перекрашивается в primary
