## Ключевые решения

### Тема
- `createThemePlugin` без кастомного адаптера → `--v0-*` на `<html>`
- `colors.ts` — единый источник (23 ключа)
- ThemeToggle: `useTheme().cycle()` + ViewTransition circle-анимация

### Vuetify0 primitives
- Button (variant × size × color + IconLoader spinner), Single (filter — inheritAttrs:false + class prop), Dialog (modal), Atom
- `storeToRefs()` для Pinia refs, actions напрямую
- Single.Root — renderless/fragment, требует wrapper `<div>` для наследования классов

### DkToggle
- `inheritAttrs: false` + explicit `class` prop для передачи на wrapper `<div>`
- `v-bind="attrs"` от Single.Item на кнопку — обязательно для клика
- Конфликт `mandatory` — убрано, иначе блокирует переключение
- Default filter: `'in-progress'` (не `'all'`)

### IconStatus (status toggle)
- `in-progress` / `done` → кликабельный checkbox (IconSquare / IconSquareCheck), toggle через `tasksStore.update()`
- `cancel` → статичный IconBan, без клика
- Использует `Checkbox` из Vuetify0 НЕ используется — просто `<button>` + Tabler icons

### Приоритеты задач (визуальная иерархия)
- **High**: `border-left: 3px solid var(--v0-error)` + `background: color-mix(error 4%, surface)` + `font-weight: 600`
- **Middle**: `border-left: 3px solid var(--v0-warning)`, обычный текст
- **Low**: `border-left: 3px solid var(--v0-muted)` + `color: var(--v0-muted)` для заголовка
- Бейдж приоритета УБРАН — бордер достаточно информативен

### Валидация
- Vuelidate + `helpers.withMessage()` на русском
- `reactive` (не computed) — v-model работает
- Title: required + minLength(3) + maxLength(200), Description: maxLength(500)

### Фильтрация
- DkToggle → computed activeFilter → storeToRefs → setFilter
- TaskList через storeToRefs → filteredTasks (reactive)

### Поиск
- pages/index.vue: инпут поиска, v-model → storeToRefs searchQuery
- searchTasks() в task-filters.ts: case-insensitive title filter
- Цепочка: filterTasks → searchTasks → sortTasks

### Сортировка
- DkSortButtons: 2 кнопки (дата / приоритет) + toggle direction (asc/desc)
- sortTasks() в task-filters.ts: PRIORITY_ORDER {low:1, middle:2, high:3}

### Централизованные иконки
- Все иконки через Tabler Icons: IconBrandGithub, IconEraser, IconLoader, IconSquare, IconSquareCheck, IconBan, IconEdit, IconTrash, IconSearch, IconChevronDown
- Inline SVG в шаблонах заменены на Tabler-компоненты

### Модалки
- DkModal: Dialog.Root + Content (без Teleport)
- Responsive: center (desktop) / bottom-sheet (mobile, smAndDown)

### Select styling
- `appearance:none`, SVG-стрелка через background-image (CSS, не inline)
- Focus: стрелка перекрашивается в primary
